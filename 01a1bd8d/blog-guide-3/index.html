Hiu c tc dng ca cc file

Nhn vo th mc src ta thy 3 th mc con

  • Components: i khi y bn s vit cc on code cho cc thnh phn ca mt trang web, v d nh header, side bar, nav bar, layout... Cc thnh phn ny s hot ng nh cc template, v s p data vo thnh page hon chnh.
  • Pages: y s l cc code ca cc trang trong page. Ch ny s vit cc trang nh Homepage, About. Cc trang post s c sinh t ng t file Markdown, nn s khng t y.
  • Images: Th mc ny l th mc cha nh, c load ln bng cch config Gatsby (ni sau). Cc nh ny l nh c nh trn trang web, v d nh nn, nh trang ch, nh trang profile... nh trong trang blog s khng t y.
